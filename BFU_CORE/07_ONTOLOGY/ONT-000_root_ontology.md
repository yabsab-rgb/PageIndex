# Root Ontology

## Purpose

This ontology defines the minimum semantic primitives required for BFU reasoning continuity.

---

# Primitive Concepts

## observer

Entity capable of state discrimination.

## state

A distinguishable configuration within an inference space.

## signal

Transferable semantic or informational variation between states.

## edge

A directional semantic relationship between nodes.

## node

An addressable semantic unit.

## canonical

A stabilized semantic structure with low contradiction pressure.

## entropy

Measure of semantic uncertainty or instability.

## collapse

Reduction of semantic ambiguity into stabilized interpretation.

## topology

Structural organization of semantic relations.

## inference

State transition generated through semantic relations.

---

# Meta Assumption

Inference continuity depends more on topology preservation than token preservation.
